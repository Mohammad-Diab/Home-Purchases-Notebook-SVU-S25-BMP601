# دفتر المشتريات المنزلية
### Home Purchases Notebook

An Android app (Java) for tracking daily home purchases with budget management, spending statistics, and theme customization.

---

## Features

- Add, edit, and delete purchases with swipe-to-reveal edit/delete actions
- Organize purchases by category with custom icons; category shown as outlined pill on each card
- Smart search bar with inline filter pills — category, date range, and text search stack independently (AND logic); backspace on empty field pops the last active filter
- Budget tracking with daily / weekly / monthly periods; home screen shows current and previous period spend
- Spending statistics with category breakdown and date-based grouping (by day / week / month / year)
- 2 modes (Light / Dark) × 4 accent colors = 8 theme combinations
- Currency support: New Syrian Pound (ل.س ج) and Old Syrian Pound (ل.س ق)
- Arabic UI with RTL layout and Arabic-Indic numerals
- Sound effects on interactions

## Tech Stack

- Java (Android)
- SQLite (local database)
- Navigation Component (single Activity architecture)
- SoundPool for audio
- No third-party UI libraries

## Requirements

- Android 7.0 (API 24) or higher
